# Junin — Ignite V80

Site responsivo para venda de pods descartaveis **Ignite V80** (8000 puffs), com entrega gratuita em Maringa/PR.

## Visao Geral

Landing page com design dark premium, fundo animado com grid e particulas, globo 3D interativo mostrando a regiao de entrega, e pedidos direto pelo WhatsApp.

## Funcionalidades

- **Hero Section** — Badge de estoque, estatisticas (12+ sabores, 8K puffs, 100% original) e globo 3D com pin em Maringa
- **Vitrine de Produtos** — Showcase interativo com troca de sabores, animacoes e especificacoes tecnicas
- **12 Sabores** — Acai Ice, Artic Gum, Frozen Mint, Menthol, Ice Mint+, Blueberry Lemon, Grape Ice, Blueberry Frozen, Strawberry Kiwi, Watermelon Ice, Lime Mango, Frozen Strawberry
- **Ofertas** — Unitario (R$90) e Combo Duplo (R$170) com badge "melhor oferta"
- **Especificacoes** — 8000 puffs, 18mL, 650mAh, 50mg nicotina, USB-C, Mesh Coil Duplo
- **Entrega** — Gratuita em Maringa/PR, Seg a Dom das 9h as 22h
- **Cookie Banner** — LGPD com opcoes de essenciais e analytics
- **Navbar** — Scroll spy, menu responsivo, efeito glassmorphism
- **Fundo Animado** — Grid estatico + particulas conexas com efeito glow

## Tecnologias

| Stack | Detalhe |
|-------|---------|
| HTML5 | Estrutura semantica |
| CSS3  | Variaveis, grid, flexbox, animacoes, glassmorphism |
| JavaScript (vanilla) | Canvas (grid + particulas + globo 3D), scroll animations, tabs |

Nao requer build ou dependencias. Basta abrir o `index.html` em um navegador.

```bash

npx serve .

# Ou abrir o arquivo diretamente no navegador
```
